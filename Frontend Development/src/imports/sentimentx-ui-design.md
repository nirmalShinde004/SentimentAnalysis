Design a modern SaaS web application UI for a platform called:

"SENTIMENTX – AI Product Review Sentiment Analyzer"

This platform allows companies to analyze customer reviews of their products using AI and NLP to determine sentiment percentages (Positive, Neutral, Negative).

Companies will register, login, enter a product URL or paste review text, and receive analytics showing the sentiment distribution of all reviews.

The design should look like a professional AI startup product similar to Stripe, Vercel, Linear, and Notion dashboards.

Use a modern dark theme with glassmorphism UI elements, soft shadows, gradient highlights, and minimal clean layout.

Font:
Inter or Poppins

Primary colors:
Indigo / Purple gradient

Background:
Dark navy or charcoal (#0f172a)

Accent colors:
Green for Positive sentiment
Yellow for Neutral sentiment
Red for Negative sentiment

Use modern spacing, rounded corners, glassmorphism cards, and data visualization components.

The layout must be responsive for desktop and tablet.
